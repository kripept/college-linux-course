# Домашнее задание 2
## Урок 2: Пользователи и права доступа

### Цель задания
Научиться управлять пользователями, группами и правами доступа в Linux.

### Задачи
1. **Работа с пользователями и группами**
   - Создайте трех пользователей: `dev1`, `dev2`, `test1`.
   - Создайте группу `developers` и добавьте в нее пользователей `dev1` и `dev2`.
   - Создайте группу `testers` и добавьте в нее пользователя `test1`.

2. **Создание структуры каталогов**
   - Создайте каталог `/projects` с подкаталогами:
       ```
       /projects
       ├── web-app
       ├── database
       └── logs
       ```
   - Настройте права доступа:
       - Каталог `web-app`: владелец `dev1`, группа `developers`, права 770.
       - Каталог `database`: владелец `dev2`, группа `developers`, права 750.
       - Каталог `logs`: владелец `root`, группа `testers`, права 755.

3. **Проверка прав доступа**
   - От пользователя `dev1` создайте файл в `/projects/web-app/` и проверьте, что `dev2` может его читать и редактировать.
   - От пользователя `test1` попробуйте создать файл в `/projects/web-app/` и зафиксируйте результат.

4. **Написание скрипта**
   - Напишите скрипт `setup-project.sh`, который автоматизирует создание пользователей, групп и структуры каталогов с правами.

### Требования к отчету
- Создайте папку `solutions/ваша_фамилия/lesson-02/`
- В папке должны быть:
    - Скриншоты, демонстрирующие права доступа (вывод `ls -l` для каждой папки).
    - Файл `setup-project.sh` с скриптом.
    - Файл `access-test.md` с описанием проверки прав доступа (какие команды выполняли и какой результат).

### Критерии оценки

   - Проверка прав доступа (2 балла)
   - Скрипт автоматизации (2 балла)
   - Отчет оформлен правильно (1 балл)
