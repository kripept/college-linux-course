# Домашнее задание 3
## Урок 3: Веб-сервер Nginx

### Цель задания
Научиться устанавливать и настраивать веб-сервер Nginx, размещать статические сайты.

### Задачи
1. **Установка Nginx**
   - Установите Nginx и убедитесь, что он запускается автоматически.

2. **Создание статического сайта**
   - Создайте каталог для вашего сайта: `/var/www/your-site/html`
   - Создайте две страницы: `index.html` и `about.html`
   - На главной странице должна быть информация о вас и вашем проекте, ссылка на страницу "О себе".
   - На странице "О себе" разместите информацию о ваших увлечениях в IT.

3. **Настройка виртуального хоста**
   - Создайте конфигурационный файл для вашего сайта в `/etc/nginx/sites-available/your-site`
   - Активируйте сайт, создав символическую ссылку в `/etc/nginx/sites-enabled/`
   - Настройте сервер так, чтобы он слушал порт 80 и отображал ваш сайт.

4. **Проверка работы**
   - Проверьте конфигурацию Nginx и перезагрузите службу.
   - Откройте сайт в браузере и сделайте скриншоты.

5. **Настройка прав доступа**
   - Убедитесь, что права на файлы сайта установлены правильно: владелец `www-data`, группа `www-data`, права 644 для файлов и 755 для папок.

6. **Дополнительное задание (необязательно, +2 балла)**
   - Настройте второй виртуальный хост для тестового сайта на порту 8080.

### Требования к отчету
- Создайте папку `solutions/ваша_фамилия/lesson-03/`
- В папке должны быть:
    - Файлы вашего сайта (копии `index.html` и `about.html`).
    - Конфигурационный файл Nginx.
    - Скриншоты работающего сайта в браузере.
    - Файл `nginx-commands.md` с перечнем команд, которые вы использовали для настройки.

   - Проверка работы и скриншоты (2 балла)
