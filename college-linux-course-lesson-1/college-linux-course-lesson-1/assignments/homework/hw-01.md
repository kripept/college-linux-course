# Домашнее задание 1
## Урок 1: Установка и первичная настройка Ubuntu Server

### Цель задания
Закрепить на практике знания по установке Ubuntu Server и базовому администрированию.

### Задачи
1. **Установка Ubuntu Server**
   - Скачайте образ Ubuntu Server 22.04 LTS с официального сайта.
   - Установите систему на виртуальную машину с следующими параметрами:
       - Память: не менее 2 ГБ
       - Диск: не менее 25 ГБ
       - Сеть: режим "Мост" (Bridge)
   - При установке создайте пользователя с вашим именем (например, `ivanov`) и дайте ему права sudo.

2. **Настройка системы**
   - После установки обновите систему:
       ```bash
       sudo apt update && sudo apt upgrade -y
       ```
   - Установите необходимые пакеты:
       ```bash
       sudo apt install -y curl wget tree htop git
       ```

3. **Работа с пользователями**
   - Создайте дополнительного пользователя `testuser` и установите для него пароль.
   - Добавьте его в группу sudo.

4. **Сбор информации о системе**
   - Определите IP-адрес сервера и запишите его.
   - Создайте файл `system-info.txt` и сохраните в него:
       - Версию Ubuntu (команда `lsb_release -a`)
       - Информацию о процессоре (`lscpu`)
       - Объем памяти (`free -h`)
       - Информацию о дисках (`df -h`)

5. **Настройка SSH**
   - Настройте подключение по SSH с вашей основной машины к серверу.
   - Сделайте скриншот успешного подключения.

### Требования к отчету
- Создайте папку `solutions/ваша_фамилия/lesson-01/`
- В папке должны быть:
    - Файл `system-info.txt` с информацией о системе.
    - Скриншот `ssh-connection.png` подключения по SSH.
    - Файл `commands-log.md` с перечнем всех выполненных команд и их выводом (где это необходимо).
